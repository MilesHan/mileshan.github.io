    初涉web3世界 - 购买第一张NFT #

    May 10, 2022 #

    事实上500rmb的购买力在web3世界里真的是洒洒水，

    然后费劲逛了好久选了一个买得起觉得还行的。

    买完，索然无味。

    1. Twitter #

    国内手机号好像是收不到验证码的。

    好不容易恢复以前忘记了的账号密码之后，再登上去，告诉我被封了，然而我一条推文或者评论都没有发过。

    然后根据流程提起申诉，反反复复，把账号搞成了永久封禁了。

    能刷推，但是没法关注评论人。

    然后用Google邮箱注册，依旧需要验证手机号。

    最后用美国信息注册的AppleId，成功注册推特。

    2. Discord #

    App操作体验堪称丝滑，但是不太符合我个人的使用习惯。

    信息的发布和聊天混在一块儿，总觉得别扭。

    国内发起的项目会用心设计这上面内容，但实际上还是会以微信群为主。

    创建 discord 的founder 大多是希望能走向国际化，可是项目的介绍和推文都无法接轨。

    后来加了几个国外的server，发现也彼此彼此。

    运营理念的问题还是运营主力选择平台的问题？

    3. 币安 #

    目前已被国内封了，无法注册。我把几年前注册的账号找了回来。

    那时买进去的100RMB 还剩52RMB。

    第一次 C2C 买了100元的ETH。第二次 400元。

    4. Metamask #

    浏览器插件让我觉得应用软件就该这样，去做PC App的话不管成本还是体验相比都感觉会有点臃肿多余。

    因为无法通过 Metamask的购买方式（需要国外信用卡）直接购买 ETH，所以只能通过 币安转账。

    币安有T+1的提现要求，C2C的资金需要24小时之后才能转出。又耽搁了几天。

    5. OpenSea #

    NFT市场。

    看上了 0.01ETH 的一个，以为很便宜了。没想到根本买不起。

    所以才有了第二次充值。

    转到metamask的时候有 0.0245ETH，以为够了吧。

    发现 Gas fee 还要差不多 3/5，得，又买不起了。

    找了个 追踪地板价的网站，从最低价一个个看合适的.

    0.011ETH，看着还行，虽然这几天都没有交易单，算了，就它吧，下单，成交。

    metamask 里还剩 0.00756ETH，目前等值 $17.26 = ¥116.17 。

    6. 总结 #

    毫无疑问，NFT是一个翻身的机会。

    但本质上，它是一个比A股更残酷的赌场。

    我看到有人通过它花了8年财富自由然后又2个月破产，有人拿1个E翻了20多倍卖出很开心。

    哪怕孤注一掷，能够赌赢的人又少之又少。对于妄想通过其翻身的人来说，和彩票没什么区别。

    它只是我通往 web3 的第一个尝试，打下一个烙印，开启一种可能。
