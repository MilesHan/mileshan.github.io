    使用 hugo 创建博客 #

    1. 安装 hugo
    brew install hugo
    1. 创建博客目录 my_blog_project
    hugo new site my_blog_project
cd my_blog_project
    1. 使用主题 hugo-book
    git init 
git submodule add https://github.com/alex-shpak/hugo-book themes/hugo-book
    1. 复制主题示例
    cp -R themes/hugo-book/exampleSite/content .
    1. 启动server查看是否成功
    hugo server --minify --theme hugo-book

    使用 GitHub Pages 发布 #

    1. Github上创建 repository

    Repository name是 github_account_name.github.io

    1. 本地博客目录下生成页面
    hugo --theme=hugo-book --baseUrl="http://github_account_name.github.io/"
    1. 将静态文件上传到 Github
    cd public
git init
git remote add origin git@github.com:github_account_name/github_account_name.github.io.git
git add .
git commit -m "first commit"
git push origin main
    1. 在 Github Actions 完成集成部署后，访问 github_account_name.github.io。完成。

    hugo 项目地址： https://github.com/gohugoio/hugo/

    hugo 中文文档： https://www.gohugo.org/doc/

    hugo-book 仓库地址： https://github.com/alex-shpak/hugo-book/
